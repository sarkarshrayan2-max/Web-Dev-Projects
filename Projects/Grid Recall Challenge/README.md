# 🧠 Grid Recall Challenge

A brain-training visual memory game built entirely with HTML, CSS, and Vanilla JavaScript. Memorize the pattern of highlighted tiles on the grid and quickly click them to reproduce the sequence. As you progress, the patterns get longer and the grid grows larger!

## 🎮 How to Play

- **Observe:** At the start of each round, a specific number of tiles will flash brightly. Pay close attention and memorize their positions.
- **Recall:** Once the tiles hide, click the exact same tiles to reproduce the pattern. (The order in which you click them does not matter).
- **Progress:** Successfully recalling the entire pattern advances you to the next level. The number of tiles you need to memorize will increase.
- **Expand:** Every 5 levels, the grid will expand (e.g. from 3x3 to 4x4, then to 5x5), significantly increasing the difficulty.
- **Lives:** You have 3 lives. Clicking an incorrect tile costs a life and forces you to retry the current level.
- **Score:** Earn points based on the level. String together perfect rounds to build your Streak Multiplier for massive scores!

## ✨ Features

- **Dynamic Grid Sizing:** Smooth CSS grid transitions between 3x3, 4x4, and 5x5 boards.
- **Visual Feedback:** Satisfying 3D flip animations and glow effects when interacting with tiles.
- **Streak System:** Rewards consistent perfect play without mistakes.
- **Local Leaderboards:** Saves your top 10 scores locally in the browser.
- **Fully Responsive:** Optimized for both desktop and mobile screens.

## 🛠️ Built With

- **HTML5:** Clean semantic markup.
- **CSS3:** Custom properties, CSS Grid, and Keyframe animations.
- **Vanilla JavaScript:** DOM manipulation, asynchronous `setTimeout` flows, and LocalStorage.

## 🚀 How to Run

1. Navigate to the `Projects/Grid Recall Challenge` directory.
2. Open `index.html` in your web browser.
3. Start training your memory!
