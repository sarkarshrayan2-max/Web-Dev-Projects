# 📸 Square Pic

Convert any uploaded image into a **perfect square format** with customizable background options. Ideal for profile pictures, thumbnails, and modern UI needs.

---

## ✨ Features
- 📷 Upload any image and convert to square
- 🌫 Blur background fill
- 🎨 Solid color fill
- 🌈 Gradient fill
- 🔍 Expansion effect (stretch edges)
- 💾 Easy preview and download
