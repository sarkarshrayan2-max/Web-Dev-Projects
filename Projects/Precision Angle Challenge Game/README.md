# 📐 Precision Angle Challenge Game

A responsive, browser-based game where players must test their spatial awareness by accurately recreating specific angles. Built with HTML, CSS, and Vanilla JavaScript using the HTML5 Canvas API.

## 🎮 Game Modes

- **Solo Challenge**: Play through 5 rounds of randomly generated target angles. Try to get the highest score possible by dragging the line as close to the target angle as you can!
- **Pass & Play (Multiplayer)**: Two players compete on the same device. Both players will receive the exact same sequence of 5 angles. Who will be the angle champion?
- **Daily Challenge**: A unique set of 5 angles generated globally every day. Everyone in the world gets the same angles to guess today. Compare your scores with friends!

## ✨ Features

- **Interactive Canvas**: Smooth, touch-friendly interactive circle for drawing angles.
- **Dynamic Scoring**: Score is calculated based on the absolute difference between your guess and the true angle.
- **Leaderboards**: Top 10 high scores for Solo and Daily modes are stored directly in your browser using `localStorage`.
- **Easter Eggs**: Can you get a **Perfect 0° Error**? Try it out and see what happens!
- **Fully Responsive**: Play comfortably on Desktop, Tablet, or Mobile.

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties for modern styling and animations.
- **Vanilla JavaScript**: Game loop, Canvas rendering, state management, and Pseudo-Random Number Generator (PRNG) for daily mode.

## 🚀 How to Run

No setup or build tools are required.

1. Clone the repository.
2. Navigate to the `Projects/Precision Angle Challenge Game` folder.
3. Open `index.html` in your web browser.
4. Select a game mode and start dragging!
