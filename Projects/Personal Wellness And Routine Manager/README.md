# 🚀 Personal Wellness & Routine Manager

A responsive and interactive Personal Wellness & Routine Manager using HTML, CSS, and JavaScript that helps users build healthy habits, maintain daily routines, and improve overall well-being.

## Features
- **Daily Routine Planner:** Track daily routines and mark them as completed.
- **Habit Tracker:** Log multiple habits and monitor your success.
- **Trackers:** Track your daily water intake, sleep hours, and exercise.
- **Wellness Insights:** Get a wellness score based on your daily progress.
- **Dark Mode Support:** Switch between light and dark modes for comfort.
- **Data Persistence:** All your wellness data is saved to your browser's local storage.

## How to use
1. Open the folder `Personal Wellness & Routine Manager`.
2. Double click the `index.html` file to open it in any modern browser.
3. Start adding your daily routines, tracking your water, and seeing your wellness score go up!

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
