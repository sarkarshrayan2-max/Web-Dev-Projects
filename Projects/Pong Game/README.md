# Pong Game

A cyberpunk Pong game built with HTML5 Canvas and vanilla JavaScript.

## Features

- Player paddle controlled by W/S or Arrow keys (touch drag on mobile).
- Segmented paddle deflection: hit position determines vertical return angle.
- Adaptive AI opponent with three difficulty tiers (Easy / Medium / Hard).
- First to 11 points wins; ball speed gradually increases per rally.
- Win/loss record persisted via `localStorage`.
- Center dashed line, neon cyan player paddle, neon pink AI paddle.

## Run it

Open `index.html` in any modern browser. Click the canvas to cycle AI difficulty.
