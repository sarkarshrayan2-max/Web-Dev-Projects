# 🌀 Path Drawing Puzzle

A memory-based puzzle game where players recreate displayed paths, scored on precision and speed, with progressively harder layouts.

## Features
- Random path generation
- Player path drawing on canvas
- Timer countdown
- Accuracy scoring based on path similarity
- Leaderboard with LocalStorage persistence
- Responsive, modern UI

## How to Run
Open `index.html` directly in your browser. No server required.
