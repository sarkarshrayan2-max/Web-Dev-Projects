# Target Reach Game

A dark, golden-themed puzzle game where players activate the correct node sequence to reach the target and complete the protocol.

## How to Play

1. Click **Initiate Link**.
2. Follow the hidden sequence of nodes.
3. Click only the correct nodes in order.
4. Reaching the target node completes the level.
5. A wrong click reboots the current sector.

## Features

- Dark gold UI with futuristic glow effects
- Grid-based node activation gameplay
- Target node objective
- Level progression with increasing challenge
- No build step required
