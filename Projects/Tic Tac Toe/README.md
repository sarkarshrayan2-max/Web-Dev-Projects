# Tic Tac Toe

A premium neon cyberpunk Tic Tac Toe game built with vanilla HTML, CSS, and JavaScript. Features both Player vs Player and unbeatable Player vs AI modes powered by the Minimax algorithm.

## Features
- **Two game modes**: PvP (pass-and-play) and vs AI (Minimax-based)
- **Neon cyberpunk UI**: Dark glassmorphic design with cyan (X) and hot pink (O) glow effects
- **Animated win detection**: Pop-in token animations, win line strike overlay, and pulsing highlight
- **Persistent scoreboard**: X wins, O wins, and draws saved to localStorage
- **Responsive**: Maintains square geometry across all screen sizes using vmin units
