# 🎓 Student Academic Planner

A responsive and interactive Student Academic Planner using HTML, CSS, and JavaScript that helps students organize their academic activities, manage study schedules, track assignments, monitor exam preparation, and improve productivity.

## Features

- **Academic Dashboard**: Overview of upcoming assignments, exams, and daily study goals.
- **Subject Management**: Add and track subjects or courses.
- **Assignment Tracker**: Track assignments with deadlines and completion status.
- **Exams Tracker**: Manage upcoming exams.
- **Study Timer (Pomodoro)**: Included Pomodoro timer for focus sessions (25 minutes).
- **Goal Tracking**: Set and track academic goals.
- **Progress Analytics**: Visual indicators for assignment completion and exam readiness.
- **Dark/Light Mode**: Easily toggle between themes.
- **Data Persistence**: Uses LocalStorage to save your data automatically.

## How to Run

1. Clone the repository.
2. Navigate to `Projects/student-academic-planner`.
3. Open `index.html` in your browser.
4. Manage your subjects, assignments, exams, goals, and use the Pomodoro timer.

## Technologies Used

- HTML5
- CSS3 (Variables, Grid, Flexbox)
- JavaScript (Vanilla, DOM manipulation, LocalStorage)
