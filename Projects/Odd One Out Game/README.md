# 🎯 Odd One Out Game

A fun logic game where users must identify the item that is different from a group of similar items.

---

## 🚀 Features
- Simple and interactive UI
- Randomized questions
- Score tracking
- Instant feedback

---

## 🧠 How to Play
1. A set of items will appear.
2. One item is different from the rest.
3. Click the odd one out.
4. Get points for correct answers.

---

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
