# 🧠 Memory Sequence Game

An interactive memory-based sequence game where players must observe and repeat an increasing pattern of colors. The game tests memory, focus, and reaction speed while providing a fun arcade-style experience.

---

## 🎮 How to Play

1. Click **Start Game**
2. Watch the sequence of highlighted colors carefully
3. Repeat the same sequence by clicking the color buttons
4. Each correct round increases the level
5. The sequence becomes longer and harder each round
6. If you click the wrong color → Game Over

---

## ✨ Features

- 🎨 Smooth animated color buttons
- 🔊 Sound feedback for each interaction
- 📈 Increasing difficulty levels
- 🏆 High score tracking (saved in browser)
- ⚡ Fast restart and responsive gameplay
- 💡 Simple and intuitive UI

---

## 🧠 Game Logic

- A random color is added to the sequence each level
- Player must repeat the full sequence in correct order
- Correct completion advances to next level
- Incorrect input ends the game
