# Endless Runner

A cyberpunk endless runner built with HTML5 Canvas and vanilla JavaScript.

## Features

- Gravity jump physics engine with single-jump restriction.
- Procedural obstacle spawning with randomized width/height.
- AABB collision detection between runner and obstacles.
- Distance-based scoring with progressive difficulty every 10 seconds.
- Screen shake and particle burst on collision.
- Green ground track with glowing neon runner and cyan obstacles.
- All-time best score via `localStorage`.

## Run it

Open `index.html` in any modern browser.
