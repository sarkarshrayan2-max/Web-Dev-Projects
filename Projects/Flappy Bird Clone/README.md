# Flappy Bird Clone

A neon cyberpunk Flappy Bird clone built with HTML5 Canvas and vanilla JavaScript.

## Features

- Gravity/velocity physics engine with flap impulse (Space / tap / click).
- Procedural side-scrolling pipe pairs with randomized gaps.
- Circle-to-rect collision detection (bird vs pipes, ceiling, ground).
- Score counter increments on pipe clearance.
- Screen shake on collision, glassmorphic Game Over modal.
- All-time best score persisted via `localStorage`.

## Run it

Open `index.html` in any modern browser.
