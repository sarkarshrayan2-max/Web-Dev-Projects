# 🚀 Digital Vision Board

A responsive productivity app to create visual goal boards with drag-and-drop cards, motivational quotes, milestones, and progress tracking.

## Features
- Add, edit, delete goals
- Drag-and-drop board layout
- Progress tracking with completion bars
- Dark/Light mode toggle
- LocalStorage auto-save
- Responsive design for desktop, tablet, mobile

## How to Run
Open `index.html` directly in your browser. No server required.
