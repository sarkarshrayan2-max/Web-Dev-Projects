# Blog

Blog is a clean, modern, and fully responsive static blog platform. It’s built with plain **HTML5**, **CSS3** (CSS custom properties, Grid, Flexbox), and **vanilla JavaScript**. No frameworks, no build steps – just open `index.html` and start reading.

The platform provides a smooth reading experience with light/dark mode, a dynamic post grid, and an elegant modal for full‑post viewing.

## Features

- 📱 Fully responsive – works on desktop, tablet, and mobile
- 🌙 Light / dark mode with local storage persistence
- 📝 Blog posts stored as a JavaScript array (easily replaceable with a CMS or static JSON)
- 🏷️ Post tags and formatted dates
- 🖼️ Lazy‑loaded images with fallback placeholders
- 🔍 Click on any post card to open a modal with full content
- ♿ Keyboard accessible modal (ESC to close) and smooth scroll navigation
- 🎨 Subtle hover animations and modern design (rounded corners, shadows, gradients)

## Tech Stack

- HTML5
- CSS3 (Grid, Flexbox, custom properties, transitions)
- Vanilla JavaScript (ES6+)
- No external dependencies
