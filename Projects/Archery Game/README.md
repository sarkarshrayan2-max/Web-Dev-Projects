# Archery Game

A cyberpunk-themed archery simulation built with HTML5 Canvas and vanilla JavaScript.

## Features

- Parabolic arrow trajectory with real-time gravity and wind drift physics.
- Click-drag pullback aiming mechanic with adjustable power.
- Concentric target scoring: Bullseye (100pts), Inner Ring (50pts), Outer Ring (20pts).
- Moving target with sinusoidal vertical oscillation.
- Three difficulty levels (Easy / Medium / Hard) affecting target speed, amplitude, wind range, and arrow count.
- Real-time accuracy tracking (hit %).
- Persistent best score via `localStorage`.
- Responsive canvas with mouse and touch input.

## Run it

Open `index.html` in any modern browser.
