# Escape Room Mini

A browser-based puzzle game where players discover clues hidden inside room objects and unlock the room using the correct code.

## Features

- Hidden clues
- Hint system
- Progress tracking
- Responsive design
- Modern glassmorphism UI

## Technologies

- HTML
- CSS
- JavaScript

## How to Play

1. Click room objects.
2. Collect clue digits.
3. Use the hint if needed.
4. Enter the correct code.
5. Escape the room.
