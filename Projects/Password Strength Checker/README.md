# Password Strength Checker

A client-side password strength analyzer built with vanilla HTML5, CSS3, and JavaScript (ES6+).

## Features

- **100% Local Evaluation** — All analysis happens in-browser via the `input` event. No data ever leaves the client.
- **5 RegEx Rule Matrix** — Minimum length (12), lowercase, uppercase, digit, and special character checks.
- **Reactive Strength Bar** — Segmented progress bar with adaptive width and color: Weak (red) → Medium (amber) → Strong (emerald) → Bulletproof (cyan).
- **Interactive Checklist** — Each rule toggles from a dim circle to a green checkmark in real time.
- **Actionable Feedback** — Dynamic recommendation text for any missing requirements.
- **Reveal/Hide Toggle** — Eye icon switches input type between `password` and `text`.
- **Copy Securely** — Clipboard API copies the password with a brief checkmark confirmation.
- **Cyber Terminal Aesthetic** — `#04060f` dark background, glassmorphic card, monospace typography, neon accent states.

## Run it

Open `index.html` in any modern browser.
