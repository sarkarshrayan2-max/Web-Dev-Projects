# Sorting Algorithm Visualizer

An interactive educational tool that visually demonstrates how sorting algorithms work step-by-step using vertical bars.

## Run it

Open `index.html` in any modern browser.

## Features

- Visualize Bubble Sort and Selection Sort algorithms in real-time.
- Customize the size of the array (number of bars).
- Adjust the animation speed during the sorting process.
- Intuitive color coding to easily distinguish between different states:
  - Blue: Default state
  - Red: Currently being compared
  - Yellow: Current minimum value (Selection Sort)
  - Orange: Actively swapping
  - Green: Sorted and in final position
- Responsive, modern card-based dark theme UI.

## Technologies Used

- Vanilla HTML
- Vanilla CSS
- Vanilla JavaScript
