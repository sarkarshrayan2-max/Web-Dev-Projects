# Startup Management Simulator

An interactive management simulation game where you step into the shoes of a CEO. Take charge of a budding startup, manage your cash flow, hire a talented team, build an amazing product, and execute marketing campaigns to scale your company to a massive valuation—all before you run out of runway!

## Features
- **Interactive Dashboard**: Track your Company Cash, Valuation, Users, and Current Month in real-time.
- **Resource Management**: Strategically allocate funds to hire Engineers and Marketers.
- **Dynamic Economy**: Engineers generate product progress, Marketers generate users and revenue.
- **Monthly Progression**: Advance time month-by-month and face the consequences of your burn rate.
- **Victory/Defeat**: Reach a target valuation to win, or face bankruptcy if cash drops below zero.

## Run It
Open `index.html` in any modern web browser to start playing. No build step or server is required.

## What I Learned
- State management using Vanilla JavaScript.
- Dynamic DOM manipulation and updating multiple UI elements simultaneously.
- Implementing a core game loop and calculating resources over simulated time periods.
- Styling complex dashboard layouts purely with CSS Flexbox and Grid.
