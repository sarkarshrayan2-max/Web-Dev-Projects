# Personal Knowledge Hub

A centralized, responsive dashboard to organize notes, bookmarks, and learning resources. Built with HTML, CSS, and vanilla JavaScript, it allows users to categorize resources, search through them, and persist data using local storage.

## Features

- **Resource Management**: Add, edit, and delete notes and bookmarks.
- **Categorization**: Group resources into predefined categories (Technology, Education, Productivity, etc.) and custom categories.
- **Search & Filtering**: Real-time search by title and filter by category.
- **Data Persistence**: All data is automatically saved to Local Storage and restored on refresh.
- **Theming**: Toggle between Dark and Light modes.
- **Responsive Design**: Adapts beautifully across mobile, tablet, and desktop screens with a modern glassmorphic dashboard layout.

## Run It

Open `index.html` in any modern web browser to start using the Personal Knowledge Hub. No installation or build steps required.
