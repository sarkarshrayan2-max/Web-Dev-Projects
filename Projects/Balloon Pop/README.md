# Balloon Pop

A neon cyberpunk balloon-popping arcade game built with HTML5, CSS3, and vanilla JavaScript.

## Features

- Procedurally spawned balloons floating from bottom to top with CSS animations.
- Three neon colors: pink, cyan, and violet with matching glow shadows.
- Click/tap to pop — triggers scale burst animation and radial particle explosion.
- Missed balloons (reaching the top) trigger a screen shake.
- Dynamic difficulty: spawn rate and float speed accelerate every 40 points.
- Score multiplier increases with difficulty.
- 45-second countdown round clock.
- High-score persistence via `localStorage`.

## Run it

Open `index.html` in any modern browser.
