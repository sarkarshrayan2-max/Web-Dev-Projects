# Pixel Color Grabber

An image color sampling tool built with HTML5 Canvas and vanilla JavaScript.

## Features

- Drag-and-drop or click-to-browse image upload (PNG, JPG, WebP).
- Canvas-rendered preview with responsive scaling.
- Click-to-sample pixel color with crosshair tracking.
- Real-time HEX, RGB, and HSL value display.
- One-click clipboard copy with visual "Copied!" feedback.
- Automatic 5-swatch monochromatic palette generation.
- Responsive layout for desktop and mobile.

## Run it

Open `index.html` in any modern browser.
