# QR Code Generator

A client-side utility built using web standard interfaces and QRious to encode string variables, links, or file parameters into crisp, downloadable QR Code images.

## Features

- **Dynamic Encoding**: Toggle instantly between link strings or local configuration files (`.txt`, `.json`, `.md`).
- **Real-time Color Shifts**: Adjust the foreground and background parameters instantly using native color components.
- **Client-side Image Compilation**: Download high-resolution PNG copies directly onto local setups using Canvas mapping.

## Setup Requirements

Served directly on standard interfaces via CDNs without dependencies. Run via root workspace web containers (`http-server`).
