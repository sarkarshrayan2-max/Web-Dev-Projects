# 🎯 Rotation Matching Game

A fun interactive browser game where players rotate a shape to match a hidden target angle. The game measures accuracy, time, and rewards higher precision with level progression.

---

## 🎮 Features

- 🖱️ Drag-based rotation control
- 🎯 Hidden target angle system
- 📊 Accuracy scoring
- ⏱️ Time tracking
- 🏆 Level progression system
- 🧠 Smart hint feedback system
- 🎮 Creative HUD-style UI

---

## 🕹️ How to Play

1. Click and drag the square to rotate it
2. Try to match the hidden target angle
3. Click **Check Score** to evaluate your performance
4. Earn higher levels by achieving good accuracy
5. Click **New Round** for a fresh challenge

---

## 📈 Scoring System

- 90%+ → 🔥 Perfect alignment + Level up
- 70%–89% → 👍 Good attempt
- Below 70% → 🟠 Keep trying
