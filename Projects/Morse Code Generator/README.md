# Morse Code Challenge

An interactive Morse code learning tool built with HTML, CSS, and JavaScript.

## Features

- Text to Morse code conversion
- Morse code to text decoding
- Morse audio playback using Web Audio API
- Decode challenge mode
- Copy Morse output
- Responsive design
- Pure Vanilla JavaScript

## Run It

Open `index.html` in any modern browser.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Web Audio API

## How To Use

### Text to Morse

1. Enter text.
2. Click **Convert to Morse**.
3. Copy or play the generated Morse code.

### Morse to Text

1. Enter Morse code using:
   - `.` for dots
   - `-` for dashes
   - Space between letters
   - `/` between words
2. Click **Decode**.

### Challenge Mode

1. Read the displayed Morse code.
2. Type your answer.
3. Submit to check.
4. Generate a new challenge.
