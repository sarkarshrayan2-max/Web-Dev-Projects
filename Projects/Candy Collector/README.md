# Candy Collector

A neon arcade candy-collection game built with HTML5 Canvas and vanilla JavaScript.

## Features

- Basket catcher controlled by Arrow keys / WASD / mouse / touch.
- Four candy types (Pink 10pts, Cyan 15pts, Purple 20pts, Gold 30pts) with increasing rarity.
- Bomb obstacles (18% spawn chance) that end your run on catch.
- Combo multiplier system — streak resets on bomb catch or missed candy.
- Floating score popups and splash particle effects on every catch.
- Screen shake and red flash on bomb collision.
- Progressive difficulty: gravity and spawn rate ramp every 40 points.
- Persistent best score via `localStorage`.

## Run it

Open `index.html` in any modern browser.
