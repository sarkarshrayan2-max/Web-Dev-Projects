# 🚀 API Explorer Dashboard

A fully responsive, interactive API Explorer Dashboard built with HTML, CSS, and Vanilla JavaScript. This application allows developers and learners to discover, test, and visualize data from various public APIs directly within the browser.

## 🎯 Objectives
- Help users understand how APIs work.
- Provide a platform for experimenting with public APIs.
- Demonstrate real-world API integration using JavaScript and the `fetch` API.
- Improve understanding of JSON data structures.
- Showcase asynchronous programming concepts.
- Create a developer-friendly learning and testing environment.

## 🌟 Key Features
- **Public API Integration:** Connect with predefined public APIs (Weather, Quotes, Users, Jokes, etc.) or add your own custom API URL.
- **API Request Panel:** Interactive controls to select methods and send requests.
- **API Response Viewer:** Display formatted, syntax-highlighted JSON responses.
- **Error Handling:** Robust error catching and user-friendly notifications.
- **Learning Section:** View details about the selected API to learn its purpose.
- **Data Persistence:** Saves recently used APIs and custom endpoints using Local Storage.
- **Developer Utilities:** Copy JSON response to clipboard.
- **Theming:** Full Light and Dark mode support.
- **Responsive Design:** Optimized for Desktop, Tablet, and Mobile views.

## 🛠️ Built With
- Semantic HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Glassmorphism)
- Vanilla JavaScript (ES6+, Async/Await)
- Phosphor Icons
- Browser `localStorage`

## 🚀 Getting Started

To run this project locally, simply clone the repository, navigate to the `Projects/API Explorer Dashboard` folder, and open `index.html` in your favorite web browser.

No build tools or external dependencies are required!

```bash
cd Projects/API Explorer Dashboard
# Open index.html in your browser
```

## 📂 File Structure
- `index.html`: The main UI scaffold.
- `style.css`: Contains all styling, responsive breakpoints, and theme variables.
- `script.js`: Handles API requests, JSON formatting, state management, and UI updates.
- `project.json`: Metadata for the repository showcase.
- `thumbnail.svg`: Visual representation of the project.
