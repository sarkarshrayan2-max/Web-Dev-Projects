# 🔺 Shape Recognition Game

A timed game where players identify hidden or distorted shapes quickly and accurately. Difficulty increases each round, rewarding both speed and accuracy.

## Features
- Random shape generation (circle, square, triangle, star, pentagon)
- Multiple choice options
- Timer countdown
- Accuracy tracking with leaderboard
- LocalStorage persistence
- Responsive, modern UI

## How to Run
Open `index.html` directly in your browser. No server required.
