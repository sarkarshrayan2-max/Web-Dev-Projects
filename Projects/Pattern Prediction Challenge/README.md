# 🔮 Pattern Prediction Challenge

A puzzle game where players predict the next item in a sequence. Procedurally generates endless puzzles and tracks accuracy across sessions.

## Features
- Procedural sequence generation (numbers, letters, mixed patterns)
- Multiple choice answers
- Round progression with increasing difficulty
- Accuracy tracking with leaderboard
- LocalStorage persistence
- Modern responsive UI

## How to Run
Open `index.html` directly in your browser. No server required.
