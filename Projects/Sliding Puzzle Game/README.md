# Sliding Puzzle Game

A 15-puzzle sliding tile game built with vanilla JavaScript.

## Features

- Solvability validation via inversion counting algorithm — guarantees every shuffle is winnable.
- Manhattan distance adjacency verification for tile movement.
- Click, touch, and keyboard (Arrow keys / WASD) input support.
- Move counter, elapsed timer, and persistent best-score via `localStorage`.
- Smooth CSS transform-based tile slide animations.
- "New Game" (shuffle) and "Reset" (solved state) buttons.
- Responsive grid with mobile-friendly touch targets.

## Run it

Open `index.html` in any modern browser.
