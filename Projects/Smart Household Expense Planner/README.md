# 🚀 Smart Household Expense Planner

A responsive and user-friendly Smart Household Expense Planner using HTML, CSS, and JavaScript that helps individuals and families effectively manage their monthly household finances.

## Features
- **Track Income & Expenses:** Easily add, categorize, and track your daily transactions.
- **Budget Planning:** Set monthly budget goals and get visual feedback on utilization.
- **Visual Analytics:** Interactive charts for expense breakdown and insights.
- **Dark Mode Support:** Seamlessly switch between light and dark themes.
- **Data Persistence:** Your financial data is automatically saved in your browser's local storage.

## How to use
1. Download or clone the project.
2. Navigate to the `Smart-Household-Expense-Planner` directory.
3. Open `index.html` in your web browser.
4. Start adding transactions and managing your budget!

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- Chart.js (for data visualization)
