# 🧩 Mini Sudoku Game

A modern, interactive Mini Sudoku game built using **HTML, CSS, and Vanilla JavaScript** with a premium glassmorphism UI and smooth gameplay experience.

---

## ✨ Features

- 🎮 Interactive 9×9 Sudoku grid
- ⌨️ Keyboard input (1–9 only)
- 💡 Hint system (auto-fills correct value)
- 🔄 Reset game anytime
- ✅ Solution checker
- 🎨 Premium glassmorphism UI
- ✨ Smooth hover and focus animations
- 📱 Responsive and clean design

---

## 🖼️ UI Highlights

- Glass-style game container
- Neon gradient title
- Smooth input glow effects
- Animated buttons with hover effects
- Clean modern Sudoku board layout

---

## 🚀 How to Run

1. Download or clone the project
2. Open folder:

Mini Sudoku Game

3. Double click:

index.html

4. Start playing 🎯

---

## 🎯 How to Play

- Fill empty cells with numbers from **1 to 9**
- Each row, column, and grid should follow Sudoku rules
- Use **Hint button** if stuck
- Use **Check button** to validate your solution
- Use **Reset button** to restart
