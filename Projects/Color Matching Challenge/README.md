# Color Matching Challenge 🎨

## Description
A web-based game where users try to match a randomly generated RGB color using sliders. The closer the match, the higher the score.

## Features
- Random target color generation
- RGB sliders for user input
- Real-time color preview
- Score based on accuracy (color distance)
- Daily challenge mode

## How to Run
Just open `index.html` in any browser. No installation required.

- HTML
- CSS
- JavaScript
