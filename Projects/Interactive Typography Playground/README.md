# Interactive Typography Playground

A modern layout design dashboard crafted for developers and designers to experiment with dynamic web font characteristics and multilayer text-shadow matrices in real-time. It enables clean text transformation visualizations and exports raw, optimization-ready CSS source blocks via an intuitive client-side sandbox.

## Run it
Open `index.html` in any modern browser.

## Features
- **Dynamic Variable Sliders:** Instant adjustments across baseline fluid layout variables (`font-size`, `letter-spacing`, `font-weight`).
- **Layered Shadow Synthesis:** Leverages vector maps to manipulate shadow blurs and offsets alongside complex stacked CSS extrusion shadows.
- **Preconfigured Design Stacks:** Includes click-to-apply matrices mimicking Cyberpunk Neon glows, solid Retro 3D pop extrusions, and chromatic Glitch layers.
- **Single-Click Code Generation:** Formats complete production-ready CSS snippet classes directly inside an active clipboard-copy listener ecosystem.
- **Zero-Dependency Core:** Implemented strictly inside native semantic scopes under 2-space strict code parameters.
